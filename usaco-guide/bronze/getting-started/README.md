# Bronze - Getting Started

## Time complexity

Generally runners on contests can handle until 10^8 operations, and maybe 5.10^8.

To calculate it we use O (big O notation) which is the worst-case time complexity of an algorithm.

Constant Factor -> We consider different operations with the same time to run (to make it easier to count), but if the time complexity is particularly tight, you may receive time limit exceeded (TLE).
